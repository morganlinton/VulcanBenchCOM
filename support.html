Keep VulcanBench measuring.

The benchmark is free and stays free. The measuring is not — and it gets more expensive with every model release.

Why this page exists The harness, the tasks, the hidden tests, and every published number are open source — reproducible for about ten dollars, and they always will be. But running each new model, at several effort levels, with repeats, is real compute, and the bill grows with every release. This is how you help pay for it: without one number going behind a paywall, and without buying anyone a better score.

Our promise

What your money buys — and what it never does.

✓

Everything stays reproducible. Every published number can be regenerated with the open harness for about ten dollars. The grading code never goes private — the moment reproduction requires paying us, the benchmark is worthless.

✓

Funding is disclosed, loudly. Sponsors and commissioners are named, never hidden. Who paid for a run appears in the run — that is the norm here, not a concession.

✓

Money buys compute and a report — never a ranking. A paid run never enters the public leaderboard unless it passes through the identical public methodology, and it is labeled as commissioned. There is no version of this where you pay for a better score.

✓

The method is open even where the tasks are not. Held-out tasks stay private so they cannot leak into training data — but the rules and the validator that grade them are public. Openness is the design; secrecy is only ever about contamination.
